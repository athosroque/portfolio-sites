Arquitetura

Medallion sobre Delta Lake

Separação rigorosa de responsabilidades: Bronze nunca recebe transformação semântica, Silver nunca carrega features de um modelo específico, e Gold é a única camada acoplada ao caso de uso — pronta para reúso por outros modelos.

Bronze

Ingestão bruta

CSV Telco carregado em Delta, imutável, sem filtros — apenas casting mínimo de tipos.

Silver

Limpeza & qualidade

Tratamento de nulos, tipagem, normalização categórica e regras de negócio aplicadas.

Gold

Features de modelagem

Features derivadas (num_services, charges_per_tenure, is_new_customer) materializadas.

# data flow
CSV (Kaggle) → Bronze → Silver → Gold → Spark ML Pipeline → MLflow → Unity Catalog (champion)

Stack técnica

Ferramentas distribuídas, não scripts soltos

Toda feature vive em transformação Spark persistida em Delta — nunca em .toPandas() intermediário. Cada execução é um run rastreado no MLflow.

Destaques de engenharia

Decisões que sustentam o pipeline

Comparação de modelos rastreada e promoção formal — aprovação no experimento ≠ aprovação para produção.

🏆

GBT como campeão

Logistic Regression, Random Forest e GBT comparados; GBT promovido por maior AUC (0.82) com tempo de treino aceitável.

🔐

Unity Catalog como gate

Nenhum modelo vira produção sem registro no Unity Catalog com alias champion. Promoção auditável, com rollback por alias.

📊

Tudo versionado no MLflow

Sem run, não existiu. Parâmetros, métricas e artefatos logados em cada execução, mesmo exploratória.
