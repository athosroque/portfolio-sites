## 📝 Resumo Executivo
Análise estatística multivariada de um dataset de e-commerce (500 usuários) para identificar padrões de conversão e segmentar clientes de alto valor. O projeto demonstra o uso do NumPy para extração de insights de negócio sem dependência de frameworks complexos de ML.

## 🛠️ Stack Técnica
- Python, NumPy, Pandas, Matplotlib, Seaborn.
- Métricas: Média, Mediana, Desvio Padrão, Matriz de Correlação.

## 💡 Insights Chave
- **Ticket Médio:** R$ 252,70.
- **Drivers de Conversão:** O tempo no site tem correlação de 0.83 com o engajamento, sendo o principal fator de retorno.
- **Oportunidade:** Redução de abandono através de ações focadas em usuários com >15 min de sessão.
