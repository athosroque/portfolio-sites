# Workflow: Revisar Outputs de um Projeto

## Uso
/revisar [slug-do-projeto]

## Passos
1. @agente-revisor verifica todos os outputs do slug informado
2. Inclui validação de consistência do portfólio (sem narrativas duplicadas)
3. Gera relatório detalhado como Artifact
4. Lista itens aprovados, avisos e erros
5. Recomenda ações corretivas se houver reprovação
