# Workflow: Status do Sistema

## Uso
/status

## Passos
1. Ler `antigravity/brain/projetos/index.md`
2. Para cada projeto: verificar se site existe, se deploy foi feito,
   se outputs estão atualizados em relação à fonte (notebook ou repo)
3. Gerar tabela de status como Artifact:

   | Projeto | Tipo | Site | Deploy | README | LinkedIn | Última atualização |
   |---|---|---|---|---|---|---|

4. Destacar projetos que precisam de update
5. Listar próximas ações recomendadas por prioridade
